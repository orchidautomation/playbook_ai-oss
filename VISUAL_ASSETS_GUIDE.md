- Clean, technical aesthetic

- Keep it minimal

## Quick Creation Guide

### 5-Minute Version (Minimum Viable Visual)

**Tool:** Canva (free)

1. Go to canva.com
2. Create custom size: 1200x1500px
3. Search templates: "Process diagram" or "Infographic"
4. Replace text with your 4 phases
5. Add timing for each phase
6. Add big stat at bottom: "480-960x faster than manual"
7. Download as PNG
8. Done.

### 30-Minute Version (Professional)

**Tool:** Figma (free)

1. Create 1200x1500px frame
2. Add rectangles for each phase
3. Add arrows between phases
4. Use consistent spacing (40px padding)
5. Font: Inter Bold for headers, Inter Regular for body
6. Colors: #0A66C2 (LinkedIn blue), #000000 (black), #F3F6F8 (light gray)
7. Add metrics card at bottom
8. Export as PNG (2x resolution)
9. Done.

### 2-Hour Version (Production Quality)

**Tool:** Figma + Illustrator

1. Create full architecture diagram
2. Add animations (for video version)
3. Create 3-4 supporting graphics
4. Professional color grading
5. Custom icons
6. Multiple export sizes
7. Carousel version (10 slides)
